## Project Description
A library wants to track the availability of its books. This system manages 25 book
records and keeps track of whether each book is currently Available, Checked Out, or
Reserved. Each record stores a Book ID, a Book Title, and a Book Status. The system also
generates a summary report describing the library's overall inventory.

Each record includes:
- Book ID
- Book Title
- Book Status

Status values:
- Available
- Checked Out
- Reserved

Summary report includes:
- Total Books
- Available Books
- Checked-Out Books
- Reserved Books

## Project Goals
- Maintain an accurate record of all 25 books in the library.
- Track each book's status as Available, Checked Out, or Reserved.
- Allow staff to add, search, display, and update book records.
- Generate a summary report showing total, available, checked-out, and reserved books.
- Practice collaborative development using GitHub (commits, pushes, shared contributions).
